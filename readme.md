## Run Locally

Clone the project

```bash
 https://github.com/shanukajain/questlab.git
```

Go to the project frontend directory

```bash
  cd questlab/medioto
```
Go to the project backend directory

```bash
  cd questlab/backend
```

Install dependencies

```bash
  npm install
```

Start the frontend

```bash
  npm run start
```

Start the server

```bash
  npm run server
```

## Tech Stack

**Backend:**  nodejs | Express | MongoDB
**Frontend:** React js
